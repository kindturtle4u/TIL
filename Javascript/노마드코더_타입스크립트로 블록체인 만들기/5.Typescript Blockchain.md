# 5.Typescript Blockchain

## 5.0 Introduction
- `npm init -y` 
- `npm i -D typescript` 타입스크립트 설치 `-D` devDependencies
- `touch tsconfig.json` 타입스크립티 설정 파일 생성
  * `"include": ["src"]`: src 폴더 전체
  * `"outDir": "build",`: complile 된 파일 디렉터리
  * `"target": "ES5"`: compile된 파일 javascript 버전
```json
{
  "include": ["src"],
  "compilerOptions": {
    "outDir": "build",
    "target": "ES5"
  }
}
```
