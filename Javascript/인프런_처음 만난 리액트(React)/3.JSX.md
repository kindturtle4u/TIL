# 3.JSX
## JSX의 정의와 역할
- Syntax Extention to JavaScript 자바스크립트 확장문법
- JSX: 자바스크립트와 XML, HTML을 합친것으로 보면 됩니다.
- JSX: JavaScript and Xml
- JSX 코드를 Javascript 코드로 변환하는 역할: React의 createElement라는 함수
```js
React.createElement(
    type,
    [props],
    [...children]
)
```
