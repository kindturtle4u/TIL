### CSS
### GRID SYSTEM
반응형, 유동형 레이아웃을 지원하는 구역 분할 시스템(12구역)

한 요소가 차지할 크기를 해상도별로 따로설정해줄수 있
- .col-lg- : 1200px 보다 클때
- .col-md- : 992px 보다 클때
- .col-sm- : 768px 보다 클때
- .col-xs- : 768px 보다 작을때

Magic Number 12 : 한 요소가 차지할 크기를 1칸에서 12칸 가지 설정해줄 수 있다.
