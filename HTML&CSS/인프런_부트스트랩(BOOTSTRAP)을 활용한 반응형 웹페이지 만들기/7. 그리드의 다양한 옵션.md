# 그리드의 다양한 옵션
Grid의 다양한 옵션 : prifix, offset, nesting, ordering

## offset (빈칸)
.col-*-offset-*

## ordering (순서를 바꾸는 ordering)
.col-*-push-*  
.col-*-pull-*

offset과 push의 차이는 offset은 뒤에있는 요소도 같이 밀림
push는 원하는 요소만 밀림

## nesting
.row 안에 또 .row를 사용하는 중첩구문

.text-capitalize : 첫번째 문자만 대문자로 만듬
