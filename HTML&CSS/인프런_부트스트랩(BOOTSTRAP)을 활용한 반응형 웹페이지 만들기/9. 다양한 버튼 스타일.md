# 다양한 버튼 스타일

.btn

.btn-default

.btn-block : 부모요소만큼 꽉찬 너비

버튼그룹
버튼을 감싸는 div를 만들고 btn-group 클래스를 추가

btn-toolbar 버튼그룹을 그룹핑함

btn-group-justified 너비가 꽉찬 형태의 버튼그룹

btn-group-vertical 버튼을 수직으로 나열

버튼을 만들기 위해서 a , button, input 태그를 사용
