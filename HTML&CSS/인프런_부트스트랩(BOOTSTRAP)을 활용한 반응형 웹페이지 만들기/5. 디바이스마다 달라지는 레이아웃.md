# 디바이스마다 달라지는 레이아웃

## Grid System > .col-*-* 여러개 사용하기

```html
<div class="col-xs-4 col-sm-6"></div> 
```

- 가로해상도 767px 이하에서는 col-xs-4가 적용되어 4/12
- 768px 이상에서는 col-sm-6이 적용되어 6/12

nth-of-type(1) : css 선택자

xs : 핸드폰 767이하
sm : 태블릿 768이상 991이하
md : 992이상

text-center : 텍스트 중간 정렬
