# 반응형 이미지 스타일
 
 .img-responsive : 화면의 크기에 따라 사이즈변경
 max-width:100%
 height : auto
 display : block;
 
 .center-block : 이미지를 중앙 정렬시에 사용
 
 img태그를 사용하지 않고 css를 활용하여 이미지를 넣으려면 div에
 .img-reponsive 클래스명을 사용하고 css에서 background-image,height와
 background-size 설정을 한다.
