## 4. CSS의 과거와 현재, 그리고 미래
- CSS는 1996년에 처음 도입되었습니다 이를 CSS 버전 1이라고 하겠습니다
- 2년 후 CSS 버전 2가 나옵니다
- 가장 최신의 버전은 CSS 버전 3으로 언젠가 버전 4가 나오게 될까요?
- 그렇지 않습니다
- CSS 버전 3에서는 CSS에 대한 개념, 접근법 개발 방식이 바뀌었기 때문입니다
- 다른 버전에 초점을 맞추는 대신 CSS는 해당 기능에 따라 구성된 몇 개의 모듈로 나뉩니다
- 텍스트의 색상을 입히는 모듈이나 그림자에 초점을 맞춘 모듈 또는 애니메이션에 초점을 맞춘 모듈 등으로 말이죠 그리고 각 모듈에 여러 버전이 있습니다
